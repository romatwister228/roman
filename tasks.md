# Задания: DOM и События

## Задание 1: Изменить текст ⭐
Создай кнопку. По клику на неё измени заголовок страницы на "Клик работает!".

---

## Задание 2: Переключить тему ⭐⭐
Создай кнопку "Сменить тему". По клику переключай класс `dark-theme` на `<body>`. Создай стили для тёмной и светлой темы.

---

## Задание 3: Счётчик кликов ⭐
Создай кнопку и число рядом. Каждый клик увеличивает число на 1.

---

## Задание 4: Поиск игрока ⭐⭐
Создай форму с полем ввода Steam ID. По сабмиту выведи введённый ID в параграф ниже. Используй `e.preventDefault()`.

---

## Задание 5: Динамический список ⭐⭐
Создай input + кнопку "Добавить". По клику добавляй введённый текст в `<ul>` как новый `<li>`. Это прототип ToDo-приложения.

---

## Задание 6: Загрузить данные с API ⭐⭐⭐
Загрузи данные героев Dota 2 с OpenDota API:
```
https://api.opendota.com/api/heroes
```

Выведи список героев (имена) на страницу. Каждый герой — отдельный `<div>`.

**Подсказка:** `fetch()` → `.json()` → цикл → `createElement` → `appendChild`

---

## Задание 7: Поиск героя ⭐⭐⭐
Добавь к заданию 6 поле поиска. При вводе текста фильтруй список героев в реальном времени (событие `input`).

---

## Задание 8: Полноценное приложение ⭐⭐⭐
Создай страницу "Dota 2 Hero Browser":
1. Загрузи всех героев с API
2. Покажи карточки с именами
3. Добавь фильтрацию по атрибуту (Strength/Agility/Intelligence)
4. Добавь поиск по имени
5. При клике на героя — покажи его данные подробнее
